# build 실패 issue

빌드 시 이전 빌드 파일이 남아 있다고 해서 경로를 읽어오지 못하는 문제가 발생했다.  
파일 이름이 변경되어도 이전 파일이 유지되는 문제였다.

```
issue.ts -> Issue.ts 로 파일명을 바꿨는데, 경로를 읽어오지 못함
```

## 원인

- 파일 이름이 해싱되지 않는 문제 : 이전 빌드 파일과 새 빌드 파일이 동일한 이름을 사용하면서, 브라우저 캐시가 엉키거나 파일 간 충돌이 발생함

## 해결

- 빌드 시작 전에 기존 빌드 파일을 모두 제거하도록 설정

1. 문제되는 파일과 dist 파일을 제거하기
2. stash해서 문제되는 현재 빌드를 stash 상태로 저장
3. build 다시하기
4. stash 제거

이렇게 잘못 빌드된 파일을 제거하고 새롭게 빌드를 적용해 경로가 엉킨 상태를 해결할 수 있다.
